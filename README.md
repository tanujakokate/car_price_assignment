# car_price_assignment# 🚗 Car Price Dataset Analysis

## 📌 Project Overview

This project analyzes a car price dataset using the Pandas library in Python. The objective is to perform basic data analysis by answering 20 analytical questions related to car prices, brands, fuel types, mileage, engine size, transmission, and other features.

## 📂 Dataset

* Dataset: Car Price Dataset
* Total Records: **10,000**
* Total Columns: **10**

### Columns

* Brand
* Model
* Year
* Engine_Size
* Fuel_Type
* Transmission
* Mileage
* Doors
* Owner_Count
* Price

## 🔍 Analysis Performed

The notebook answers the following questions:

1. Average price of all cars.
2. Brand with the highest average price.
3. Most common fuel type.
4. Number of cars with mileage greater than 200,000.
5. Distribution of cars by transmission type.
6. Car model with the highest price.
7. Average price by fuel type.
8. Number of cars manufactured in 2020 or later.
9. Average engine size for each brand.
10. Car with the lowest mileage.
11. Correlation between mileage and price.
12. Number of cars with more than 2 owners.
13. Average price by manufacturing year.
14. Car with the highest number of doors.
15. Average mileage by fuel type.
16. Number of Toyota cars.
17. Average price of cars with automatic transmission.
18. Car with the highest owner count.
19. Average engine size of diesel cars.
20. Number of cars priced above $10,000.
